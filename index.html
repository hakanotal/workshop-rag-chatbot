What we’ll build

A full Agentic RAG pipeline over Mary Shelley’s Frankenstein, end-to-end:

  • Local LLM chat with Ollama (Llama 3.2)
  • Document chunking, embedding, and a vector database (ChromaDB)
  • Simple RAG chain with strict grounding (LangChain)
  • Agentic RAG: Retriever + Verifier agents (CrewAI)
  • Shareable chat UI (Gradio)
  • Head-to-head comparison: Plain LLM vs Simple RAG vs Agentic RAG
